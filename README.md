# mitsa-web
A repo for deployment of all event related webpages .

## Industry Night 2026 Deployment

The `industry_night_2026` folder is deployed with GitHub Actions to GitHub Pages.

One-time GitHub setup:

1. Open the repository on GitHub.
2. Go to `Settings` -> `Pages`.
3. Set `Source` to `GitHub Actions`.
4. Push changes to `main`, or run `Deploy Industry Night 2026` manually from the `Actions` tab.
